# Changelog

Tous les changements notables de ce projet seront documentés dans ce fichier.

Le format est basé sur [Keep a Changelog](https://keepachangelog.com/fr/1.0.0/),
et ce projet adhère au [Semantic Versioning](https://semver.org/lang/fr/).

## [1.0.0] - 2026-02-14

### Ajouté
- Version initiale du package
- Téléchargement de segments YouTube en MP4
- Interface en ligne de commande (`yt-segment`)
- Support des formats de temps MM:SS et HH:MM:SS
- Validation des URLs YouTube
- Qualité vidéo maximale disponible
- Documentation complète
- Tests unitaires de base

### Fonctionnalités
- Commande CLI globale après installation
- Utilisation programmatique en Python
- Exécution en tant que module Python
- Gestion des erreurs robuste
- Messages de progression détaillés
